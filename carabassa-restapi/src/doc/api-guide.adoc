[[overview]]
= Overview

[[overview-http-verbs]]
== Introduction
This guide defines all Carabassa RESTful API resources.

The goal is to provide an extensible image manager using REST API to organize, index and search your images.

The images are grouped in _datasets_, i.e, a _dataset_ is an images collection.
